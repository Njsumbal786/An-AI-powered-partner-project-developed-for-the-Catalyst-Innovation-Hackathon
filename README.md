 Installation & Setup

To run the NJ Blogs project locally, first download or clone the source code from the GitHub repository. After downloading, extract the files (if downloaded as a ZIP) and open the project folder.

 Desktop (Laptop / PC)

This project is built using basic web technologies and does not require any special installation. Open the main HTML file (for example, `index.html`) in any modern web browser such as Chrome, Edge, or Firefox. The application will run directly in the browser. For a smoother experience, the project can also be opened using a local development server like Live Server in VS Code, but this is optional.

Mobile (Android / iOS)

To view the project on a mobile device, transfer the project files to your phone or upload them to any hosting or file-sharing service. Then open the main HTML file using a mobile web browser such as Chrome (Android) or Safari (iOS). The application will display in the mobile browser and can be interacted with normally. For best results, ensure the browser allows local file access or use a hosted version of the project.
